🚀 Setup Instructions
1. Environment Configuration
Create a file named .env in the root directory. This file is used to store sensitive credentials and should never be committed to version control.

Copy the template below into your .env file:

Code snippet
```
# Database connection string
DATABASE_URL=<Contact us for the db URL>

# OpenAI API Key
OPENAI_API_KEY= <Insert openai api key here>
```

2. Request Access
The database and API keys are restricted for security. Please contact us for the database URL

3. Run the Application
Once the .env file is ready, install dependencies and start the development server:

```
Bash
npm install
npm run dev
```
